# Excel進捗管理表 クレンジング自動化ツール

## 概要
現場でよくある「多段ヘッダ・表記ゆれ・数値/文字列混在」の
Excel進捗管理表を、Pythonで安全にクレンジングするツールです。

- 元データは変更せずバックアップを作成
- 自動で整形できない箇所は赤字で可視化
- ピボットテーブル作成に適した形式へ変換

PMO業務で実際に発生した課題を元に作成しました。

---

## 想定課題（Before）
- 担当者名の表記ゆれ（TES山田 / TES_山田 / QX斎藤 等）
- 工数（分）が空欄・文字列・数値混在
- 日付フォーマットがバラバラ
- 多段ヘッダで集計しづらい
- 数式IDが行削除で崩れる

---

## 対応内容（After）
- ヘッダを1行に統合
- 担当者名を「会社コード_氏名」に正規化
- 会社コード不明な場合は赤字で表示（誤変換防止）
- 工数（分）→ 工数（時間）列を追加（[h]:mm形式）
- ID列を値貼り付けに変換
- 最終行以下を削除しピボット対応

---

## 使用技術
- Python 3.x
- openpyxl
- pathlib
